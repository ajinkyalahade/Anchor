# Rewards Decisions

- XP is granted on completion, not performance alone.
- Focus has a slightly higher XP weight than Word Gym.
- Calm earns XP to reinforce regulation effort.
- Missed days activate recovery state rather than deleting streak progress.
- Comeback bonus is consumed once on the next rewarding action.

## Deferred

- Full theme application for every unlocked visual theme.
- User-facing unlock inventory management.
