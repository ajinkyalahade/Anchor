# Calm Zone

Calm Zone contains regulation tools: breath coach, grounding, RSD interrupt, crisis safety, and
the anxiety spiral stop.

## Owned Code

- Frontend: `frontend/src/pages/CalmPage.tsx`
- Components: `BreathCoach`, `AnxietySpiralStop`
- Backend: `backend/app/api/calm.py`
- Safety: `backend/app/ai/safety/classifier.py`
- Prompt: `backend/app/ai/prompts/rsd.py`

## User Value

The module gives the user immediate down-regulation tools without requiring setup or analysis.
