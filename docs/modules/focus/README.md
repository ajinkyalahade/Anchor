# Focus

Focus helps the user start and sustain one task with ADHD-tuned Pomodoro presets, a visual timer
bar, task decomposition, and distraction capture.

## Owned Code

- Frontend: `frontend/src/pages/FocusPage.tsx`
- Components: `TaskDecomposer`, `TimerBar`, `DistractionPark`, `VoiceInput`
- Backend: `backend/app/api/focus.py`
- AI: `backend/app/ai/router.py`, `backend/app/ai/prompts/decompose.py`

## User Value

The feature reduces task-initiation friction by making the first step small and visually anchoring
time.
