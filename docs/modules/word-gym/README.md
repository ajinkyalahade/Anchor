# Word Gym

Word Gym is a 60-second word association game for retrieval warm-up and cognitive state shifting.

## Owned Code

- Frontend: `frontend/src/pages/GamesPage.tsx`
- Backend: `backend/app/api/games.py`
- AI: Gemini route in `backend/app/ai/router.py`
- Prompt: `backend/app/ai/prompts/wordgym.py`

## User Value

The game creates a short, rewarding dose that exercises word retrieval without framing it as a
lesson.
